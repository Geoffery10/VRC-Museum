## Art Requirements
### Suported Formats
* JPEG
* PNG (Transparentcy Allowed but Not Recommended)
* GIF (Not Quest Compatible)

### Image Size
* 2048px * 2048px or smaller
* 1:1 Aspect Ratio Prefered but Not Required
